## 📌 Overview
**VoteSahayak** is an interactive, step-by-step assistant designed to help Indian citizens understand the election process. From voter registration to locating polling booths, the platform empowers voters with accurate information, interactive tools, and an AI-powered assistant.

## ✨ Key Features

*   **Voter Journey:** An interactive, step-by-step guide covering eligibility, Form 6 registration, voter list verification, and polling day instructions.
*   **Polling Station Locator:** Interactive district-level maps (Kolkata, Howrah, South 24 Parganas, North 24 Parganas, Hooghly) to find polling booths effortlessly.
*   **AI Chat Assistant:** Powered by Groq's Llama 3.1 model, offering instant, context-aware answers to election-related queries via a modern dark-themed UI.
*   **Knowledge Quiz & Certification:** A 5-question interactive quiz that instantly scores users and generates a downloadable PDF "Informed Voter" certificate.
*   **Election Timeline & Helpline:** Tracks key election dates and provides direct access to ECI contact information and FAQs.

## 🛠️ Tech Stack

*   **Frontend & Framework:** Streamlit
*   **Mapping Interface:** Folium
*   **AI Integration:** Groq API (OpenAI-compatible)
*   **Certificate Generation:** jsPDF
*   **UI/UX:** Custom Dark Theme with Indian tricolor accents (Saffron `#FF9933` & Green `#046A38`)
